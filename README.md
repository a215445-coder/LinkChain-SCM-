# LinkChain-SCM-
服务制造业仓储采购的供应链管理系统，包含采购、库存、供应商、仓储调拨、数据统计等模块，实现多仓联动与订单全链路追踪。
# LinkChain SCM供应链协同平台
面向制造业的供应链管理系统，包含采购、库存、仓储、数据可视化。

## 技术栈
- Vue3 + Pinia
- Element Plus
- ECharts 数据大屏
- AntV X6 拓扑图
- WebSocket 实时消息

## 功能模块
- 采购订单管理
- 库存与多仓库管理
- 供应商管理
- 仓储流转可视化
- 实时预警与消息推送
- 权限控制系统

## 项目说明
个人前端核心开发项目，负责业务页面、状态管理、可视化与实时通信功能。
